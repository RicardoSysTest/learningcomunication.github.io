# El Rapport Como Herramienta para Conectar con el ritmo del Otro

**Rapport** dos o mas personas que sienten en sintonia o confianza cuando tenemos cosas en comun, sintonia psicologica. ¿Con quien podemos iniciar una convesasion? Es un fenomeno que nos permige generar confianza con una persona. Cuando somos escuchas buscamos que la persona que esta emitiendo mensaje buscamos hacer rapport buscar similitud para generar un sesacion psicologica de confianza. 

## ¿Que podemos igualar para generar Raport?

1. Estilo de lenguaje. (muletilas)
2. Patrones de lenguaje (Ejemplos y Anecdotas)
3. Imagen. 
4. Posiciones fisicas.
5. Patrones de respiracion o parpadeo.
