# Mejorar nuestra forma de Escuchar.

## ¿Como mejorar nuestra capacidad de escuchar?

Un buen receptor, puede correr el mensaje del emisor cuando este no es claro. Si quien recive el mensaje no esta listo para atender el mensaje por muy bueno que sea el mensaje no podremos recivir el mensaje

Los mejores *EMISORES* distingen el tipo de comunicacion que emplearan segun lo que quieren transmitir y lograr. **SE COMUNICAN ESTRATEGICAMENTE**

Los mejores ecuchas, conectan con el ritmo, con el estado y con el contenido de quien emite el mensaje.
