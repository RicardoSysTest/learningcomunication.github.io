# Las Respuestas. 

* La peor respuesta es la vaguedad (ambihuedad)
* 5 Respuestas positivas:
    1. Aceptacion clara ().
    2. Decir que No, y hacer responsable de las consecuencias. 
    3. Solicitar mayor claridad: preguntar cualquier cosa que no haya quedado clara. 
    4. Solicitar un mayor plazo para comprometerse: necesito verificar que existen las condiciones adecuadas para lo que me estas solicitando
    5. Renegociar el compromiso, haciendo una contra-oferta. 


## Agradecimineto efectivo: (cierre)

* Señalar e indentificar que si se cumplio.
* REconocer lo bien hecho
* Recordar que podemos hacer acuerdos futuros.
